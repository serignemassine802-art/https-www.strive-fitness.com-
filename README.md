# Strive Fitness - Sitio Web Oficial

Sitio web moderno y profesional para Strive Fitness Club en Thatcher, Arizona.

## 🚀 Tecnologías Utilizadas

- **Next.js 14** - Framework React
- **TypeScript** - Tipado estático
- **Tailwind CSS** - Estilos modernos
- **Vercel** - Hosting

## 📋 Características

- ✅ Página de inicio atractiva (Hero section)
- ✅ Sección de servicios
- ✅ Información sobre la empresa
- ✅ Formulario de contacto funcional
- ✅ Responsive design (móvil, tablet, desktop)
- ✅ Navegación intuitiva
- ✅ Optimizado para SEO

## 🏃 Instalación y Ejecución

### Requisitos
- Node.js 18+
- npm o yarn

### Pasos

1. **Clonar el repositorio**
```bash
git clone https://github.com/serignemassine802-art/https-www.strive-fitness.com-.git
cd https-www.strive-fitness.com-.git
```

2. **Instalar dependencias**
```bash
npm install
```

3. **Ejecutar en desarrollo**
```bash
npm run dev
```

4. **Abrir en el navegador**
```
http://localhost:3000
```

## 🔧 Comandos Disponibles

- `npm run dev` - Ejecutar servidor de desarrollo
- `npm run build` - Compilar para producción
- `npm start` - Iniciar servidor de producción
- `npm run lint` - Verificar código
